# EcoConnex

EcoConnex is an open-source e-commerce website for electric vehicle (EV) spare parts. It provides a responsive shopping experience with product browsing, search, filtering, shopping cart, and WhatsApp order integration.

## 🚀 Live Demo
https://ecoconnex.in

## ✨ Features
- Responsive Design
- Product Search
- Category Filters
- Shopping Cart
- WhatsApp Order Integration
- SEO Optimized
- GitHub Pages Deployment

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
